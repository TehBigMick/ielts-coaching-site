课程价格

我提供灵活的课程套餐，让您根据需要选择最合适的服务。以下是我提供的各类课程和服务的详细价格信息。

试听课程

首次尝试课程，了解我的教学方法和反馈方式。

￥300 / 60分钟

通过试听，您可以确定这是否是适合您的学习方式。如果决定继续，可以选择课时套餐。

写作 Task 1 辅导

针对 Task 1，帮助您提升概述和数据分析技巧。

￥500 / 60分钟

通过专项训练，您将学会如何精确分析数据，并用恰当的语言组织表达。

写作 Task 2 辅导

针对 Task 2，提升您的逻辑结构和语言准确度。

￥500 / 60分钟

帮助您理解评分标准，提升任务响应和论证的清晰度。

口语辅导

提高您在口语考试中的表现，重点是流利度和词汇的多样性。

￥500 / 60分钟

通过一对一的口语训练，帮助您清晰有条理地表达思想，并减少重复。

全套雅思备考

为考生提供全方位的备考服务，包括写作、口语、听力和阅读。

￥2500 / 10课时

此套餐包括定制化的课程安排，帮助您在所有科目上提升，并进行模拟考试。

写作批改

仅提供写作批改服务，专注于提高您的写作分数。

￥300 / 每篇

您可以提交 Task 1 或 Task 2 作文，我会为您提供详细反馈，并指出提高的方向。

立即预约课程
